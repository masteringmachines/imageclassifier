## Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/YOUR_USERNAME/image-classifier.git
cd image-classifier
```

### 2. Create a virtual environment

```bash
python -m venv venv
source venv/bin/activate      # Windows: venv\Scripts\activate
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

> PyTorch install may vary by platform. See [pytorch.org](https://pytorch.org/get-started/locally/) for CUDA-specific installs.

### 4. Download ImageNet class labels

```bash
python download_labels.py
```

### 5. Run the app

```bash
python app.py
```

Open your browser at **http://localhost:5000**

## How It Works

1. User uploads an image via the browser
2. Flask receives the file and opens it with Pillow
3. The image is preprocessed (resize â†’ center crop â†’ normalize)
4. MobileNetV2 runs a forward pass and returns 1000 class logits
5. Softmax converts logits to probabilities
6. Top 5 predictions are returned as JSON and rendered in the UI
